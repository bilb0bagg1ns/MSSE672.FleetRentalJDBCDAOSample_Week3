- Steps to run the JDBCDAOSample


1. Assumes MySQL is installed and running.
     - go to MySL bin folder and run 'mysqld'
2. Run the sql in CreateTable.sql - to create the table and insert it with data.
3. Edit bin\mysetenv.cmd and update the environment.
3. run bin\mysetenv.cmd
4. run the TestDriver: ant runTestDriver
